1. Устанавливаем nodeJS с https://nodejs.org/en/ - мэнэджер пакетов для инсталяции необходимого энвайрмента из командной строки
2. Устанавливаем angularCli - утилита для сборки проекта и деплоя проекта
из командной строки
npm install -g @angular/cli
для проверки что все установилось следующая команда должна показать версию 
ng --version 
3. Запуск сервера из папки **\gloria\gloriaUI\
В командной строке
ng serve
4. В браузере проект можно открыть по следующему адресу http://localhost:4200/
